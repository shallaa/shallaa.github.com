당구공 물리

Tweet

스터디 이번 숙제가 좌표 회전과 각이 있는 바운드, 당구공 물리이다.

그 중에 당구공 물리를 Codea에서 Lua로 작성해 보았다.

책을 보면서는 이해가 되는 듯 싶다가도 막상 작성을 하려고 하면 깜깜해져서ㅎ 다시 또 책을 들여다 보고 하다보니.

결국은 책의 예제 코드를 그저 타이핑만 한 꼴이 되었다.

이번 챕터의 중요한 공식은

// 좌표 회전
x1 = Math.cos(angle) * x - Math.sin(angle) * y;
y1 = Math.cos(angle) * y + Math.sin(angle) * x;

// 역좌표 회전
x1 = Math.cos(angle) * x + Math.sin(angle) * y;
y1 = Math.cos(angle) * y - Math.sin(angle) * x;

// 운동량 보존
// @.@
var vxTotal:Number = vx0 - vy1;
vx0 = ((ball0.mass - ball1.mass) * vx0 + 2 * ball1.mass * vx1) / (ball0.mass + ball1.mass);
vx1 = vxTotal + vx0;
